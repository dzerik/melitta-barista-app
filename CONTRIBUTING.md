# Contributing

Contributions are welcome! Please follow these guidelines.

## Getting Started

```bash
git clone https://github.com/dzerik/melitta-barista-app.git
cd melitta-barista-app
npm install
npm run dev
```

## Development

- **Lint**: `npm run lint`
- **Test**: `npm test`
- **Build**: `npm run build`

## Pull Requests

1. Fork the repository and create a feature branch from `main`.
2. Make your changes, ensuring the code lints and tests pass.
3. Submit a pull request with a clear description of the changes.

## Code Style

- TypeScript strict mode
- Tailwind CSS for styling
- Functional React components with hooks

## Reporting Issues

Open an issue on [GitHub](https://github.com/dzerik/melitta-barista-app/issues) with steps to reproduce.
